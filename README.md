# GOB2
A comprehensive year-by-year narrative dashboard of Barbados' financial accountability journey through 24 years of Auditor General's reports. Features interactive visualizations, special audits, and cultural context through Bajan proverbs.
# README.md

```markdown
# 🇧🇧 Barbados Financial Accountability Dashboard 2003-2026

**24 Years. 3 Acts. One Story.**

An interactive Streamlit dashboard that tells the complete story of Barbados' financial accountability journey through 24 years of Auditor General's reports.

---

## 📊 The Story in 3 Acts

| Era | Years | Opinion | Debt Range |
|-----|-------|---------|------------|
| 🟢 The Golden Years | 2003-2007 | ✅ Clean | $3.9B - $5.0B |
| 🟡 The Slow Decline | 2008-2017 | ⚠️ Disclaimer | $5.7B - $13.1B |
| 🔴 The Breaking Point | 2018-2023 | ❌ Adverse | $13.2B - $14.9B |
| 🟣 The Path Forward | 2024-2026 | ⏳ Pending | $14.9B - $15.2B |

---

## 🎯 Key Features

- 📖 **"The Full Story" View** - Narrative arc with Acts and Chapters
- 📌 **Year-at-a-Glance** - Detailed metrics for each year
- 💰 **Financial Metrics** - Revenue, Expenditure, Deficit, Debt
- 🔍 **Special Audits** - Complete list of special audits per year
- 📈 **Interactive Charts** - Revenue composition, expenditure breakdown, historical trends
- 🇧🇧 **Bajan Proverbs** - Cultural context with meanings

## 📂 Installation & Setup

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/barbados-accountability-dashboard.git
cd barbados-accountability-dashboard
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Run the application**
```bash
streamlit run barbados_dashboard.py
```

4. **Open your browser** at `http://localhost:8501`

## 📄 Data Source

This dashboard is built entirely from publicly available data from:

**Auditor General's Reports of Barbados** (2003-2024)
- Annual reports on government financial statements
- Special audit reports
- Performance audit reports
- Financial year data

The data has been compiled, structured, and enriched with cultural context to tell the complete accountability story of Barbados.

---

## 📊 Financial Data Overview

### Key Metrics Tracked
- **Revenue**: Government income (Billions $B)
- **Expenditure**: Government spending (Billions $B)
- **Deficit/Surplus**: Revenue minus Expenditure (Billions $B)
- **Public Debt**: Total government debt (Billions $B)
- **Debt-to-GDP Ratio**: Debt as percentage of GDP (%)

### Complete Dataset (2003-2026)
- 24 years of financial data
- 4 audit opinion eras
- 50+ special audits documented
- 100+ key issues tracked
- 24 Bajan proverbs with cultural meanings

## 🔑 Key Insights from the Data

| Insight | Detail |
|---------|--------|
| **SOE Consolidation** | Still not done after 15+ years |
| **Unverified Tax Receivables** | $2.43B flagged as unverified |
| **Pension Liabilities** | Hidden from balance sheets |
| **Asset Discrepancies** | $719M in unverified assets |
| **Consecutive Adverse Opinions** | 6 years (2018-2023) |

### Bajan Proverb of the Data Set
> *"When de bottom drop out, e hard to patch it."* (2023)
> 
> *Meaning: When the foundation is broken, fixing it is extremely difficult.*
